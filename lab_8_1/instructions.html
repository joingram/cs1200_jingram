2.1. function name: replicate

This function takes one string parameter, and one integer parameter and returns a list where the provided string occurs n times.

Examples:
replicate('h1', 2) will return ['hi','hi']
replicate('hello', 3) will return ['hello','hello','hello']



2.2 function name: unique

This function takes arbitrary number of strings as parameter, and returns a list where all duplicate values have been removed. Order of items in the returned list does not matter.

Examples:
unique('blue','green','blue','red') will return ['blue','green','red']
unique('apple','banana','apple') will return ['apple','banana']
